# nvpress综合示例主题

## 介绍

这是一个基于`vue3+nvpress`的示例主题。

## 安装

```bash
# 克隆项目
git clone https://github.com/iscottt/base-theme.git

# 进入项目目录
cd base-theme

# 安装依赖
npm install

# 启动项目
npm run dev
```

## 功能

- [x] 顶部导航栏
- [x] 文章列表
- [x] 文章详情
- [x] 文章标签
- [x] 文章分类
- [x] 文章归档
- [x] 文章评论
